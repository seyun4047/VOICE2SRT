# VOICE2SRT

This project converts speech to text<br>
and generates subtitle files by splitting into SRT-format timestamps in seconds.

## Dependencies
This project depends on the OpenAI Whisper module.

```bash
pip install openai-whisper
brew install ffmpeg # for mac
```
